BRLLNT - Go Deep

Design Concept

This was one of the first music visualizers I had ever made. I wanted to make something that would help me get started in figuring out how Trapcode Soundkeys worked. I picked this song since it has a fair amount of contrast and has plenty of different sounds to react to. There wasn't a particular design style I was going for, but I was inspired by Shaped Studios's tutorial on Soundkeys to make something more abstract.

Software Used

The software I used to achieve this look is purely Trapcode and Adobe After Effects.

Process

I first created an abstract object using Trapcode Tao, then made it somewhat trancelucent, then set it to spin and deform based on the bass drum in the song. I also added a light source close to the object to have it stand out from the background. After that, I added a patterned background using Trapcode Mir and set it to react to the general vocal range. I also added another source of light in the background and set it to the same ranges as the Mir object.
